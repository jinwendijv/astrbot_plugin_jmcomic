# JMComic AstrBot 插件

将 JMComic-Crawler-Python 封装为 AstrBot v4.x 可用的插件，支持漫画搜索、查看详情、自动生成PDF并发送、群白名单权限控制。

## ✨ 功能特性

- 🔍 **漫画搜索** - 按关键词搜索禁漫漫画
- 📖 **详情查看** - 查看漫画详细信息
- 📄 **PDF自动生成** - 下载后自动合并为PDF发送到群/私聊
- 🏆 **排行榜** - 查看日/周/月排行榜
- 🧹 **缓存清理** - 一键清理下载的图片和PDF
- 🔒 **群白名单** - 控制哪些群可以使用插件
- 👤 **账号登录** - WebUI配置JM账号密码，支持登录下载
- ⚙️ **可视化配置** - AstrBot WebUI完整配置界面

## 📦 安装

### 方式一：手动安装（推荐）

1. 下载 `astrbot_plugin_jmcomic.zip` 并解压到 AstrBot 的 `data/plugins/` 目录

2. 安装依赖：
```bash
pip install jmcomic>=2.7.0 pillow>=10.0.0 pyyaml>=6.0
```

3. 重启 AstrBot 或在 WebUI 重载插件

### 方式二：WebUI安装

在 AstrBot WebUI 插件市场中搜索并安装

## ⚙️ 配置（WebUI可视化界面）

插件首次启动后，在 **AstrBot WebUI → 插件配置** 中进行设置：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `启用群白名单` | 开启后仅白名单内的群可以使用插件 | `false` |
| `群白名单列表` | 允许使用插件的群ID，每行一个 | `[]` |
| `自动清理临时文件` | 发送PDF后自动删除下载的图片缓存 | `true` |
| `JM账号` | JM邮箱/用户名，登录后可下载需要权限的漫画 | 空 |
| `JM密码` | JM账号密码（敏感字段，隐藏显示） | 空 |
| `JM站点域名` | 可切换为 18comic.vip / 18comic.org | `18comic.vip` |

## 📝 命令列表

### 普通用户命令

| 命令 | 功能 | 示例 |
|------|------|------|
| `/jm搜索 [关键词] [页码]` | 搜索漫画 | `/jm搜索 全彩 1` |
| `/jm详情 [漫画ID]` | 查看漫画详情 | `/jm详情 123456` |
| `/jm下载 [漫画ID]` | 下载并生成PDF发送 | `/jm下载 123456` |
| `/jm排行 [日/周/月]` | 查看排行榜 | `/jm排行 周` |
| `/jm帮助` | 显示帮助 | `/jm帮助` |

### 管理员命令

| 命令 | 功能 | 示例 |
|------|------|------|
| `/jm清理 [类型]` | 清理本地缓存文件 | `/jm清理 全部` |

**清理类型可选：**
- `全部` 或 无参数：清理图片+PDF（默认）
- `图片` 或 `img`：只清理下载的图片
- `pdf` 或 `PDF`：只清理生成的PDF

## 🎯 使用示例

### 1. 搜索漫画
```
/jm搜索 全彩
```

### 2. 下载并发送PDF
```
/jm下载 123456
```
- 下载完成后自动生成PDF
- 直接发送到当前聊天
- 无尺寸限制，支持 webp/png/jpg/gif 所有格式
- ✅ 支持重复下载（即使已下载过也能重新生成PDF）

### 3. 查看排行榜
```
/jm排行 周
```

### 4. 清理缓存
```
/jm清理 全部
```

## 🔒 权限说明

- **私聊**：不受白名单限制，所有用户都可以使用
- **群聊**：
  - 未启用白名单：所有群都可以使用
  - 已启用白名单：只有白名单内的群可以使用
- **管理员命令**：仅Bot管理员可以使用

## 🐛 常见问题

### Q: 提示 "JMComic客户端未初始化"
A: 执行 `pip install jmcomic>=2.7.0` 安装依赖

### Q: 提示 "Pillow未安装"
A: 执行 `pip install pillow` 安装依赖

### Q: PDF发送失败
A: 确保使用 AstrBot v4.x 最新版本，插件使用官方标准API

### Q: 某些漫画下载失败
A: 在WebUI配置JM账号密码后重试，部分漫画需要登录权限
